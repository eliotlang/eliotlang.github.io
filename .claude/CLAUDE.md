# Claude Memory

## Project Overview

This is the GitHub website project of ELIOT. 
ELIOT is a functional, generic programming language for microcontrollers, implemented in Scala 3, located
in a different repository.

The homepage is a standard GitHub Pages project, partly static web pages, partly content in Markdowns
generated by Jekyll automatically.

## Relationships

The ELIOT compiler itself is available locally at "../eliot", with its own Claude configuration and skills,
example code and various other artefacts. This might be needed to understand the language itself, or to
generate and test examples.

The design is available in Claude Designer. That should be contacted to get design guidelines, artwork
and other materials.
