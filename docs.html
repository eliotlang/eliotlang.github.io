A hands-on guide to Eliot — the functional, generic language for microcontrollers. It starts from a one-line program and builds up to the type system, algebraic effects, abilities, and the layer model, one small idea at a time. Every program on these pages is real Eliot that compiles and runs on the JVM today.

New here? Start with the introduction and work forward — each chapter assumes the ones before it. Already know the basics? Jump straight to the part you need from the sidebar.

Start reading →

Eliot is pre-release (alpha). The core language described in the first parts of this guide is stable and runnable today; chapters marked soon cover areas that are still settling, and are outlined so you can see where the guide is going.
