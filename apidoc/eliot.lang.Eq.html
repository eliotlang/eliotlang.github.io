Eq

ability Eq[A]

Equality — the ability that supplies == and != for a type A.

Because Eliot draws no line between the type level and the value level, Eq works for ordinary values and for types: an instance for the type of all types (Eq[Type], structural equality of types) lets a compiler-evaluated guard such as where E1 != E2 decide whether two type parameters are the same. Value instances (Eq[Int], Eq[String], …) are added by the platform layers where a representation exists.

equals is the single primitive; == and != are the operators built on it.

Equality operator: a == b is true when a and b are equal under Eq[A].
